# Projekt
## Unabdingbare Projektkriterien
* Strukturübergreifend
* Ziel
* Ressourcen beschränkt
* Termine, Endetermin
* definiertes Ende
* formale Übergabe

## Schwachstellen von Projekten
* hohe Komplexität
* Intransperenz
* keine festen Budgets
* keine Erfolgsgarantie
